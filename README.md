# ShodanBlocker
###### Blocks Shodan IPs From Scanning Your Servers.

*Bought an OVH and got given user `ubuntu` etc? Here's how to fix that issue.*

## Getting Started
* Install NodeJS
* Install NPM
* Execute `node shodan.js` 
* Enjoy Shodan.io Scanners being blocked

Hope it works for you.
